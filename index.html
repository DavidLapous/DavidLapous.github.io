I am currently a PostDoc in the GeomeriX team at Inria (Saclay, Polytechnique campus).

I am working on Topological and Geometrical Data Analysis (TDA), with a particular emphasis on the links between multiparameter topological persistence theory and machine learning.

Most of my work can be viewed in pictures on the website of my library multipers.
You can install it in python via 

  >>> pip install multipers
